PathAttribute option

Gets or sets the attribute inside an HTML tag which references the external file. 

  public string PathAttribute { get; set; }

The default value is src.

Configuring PathAttribute

The PathAttribute option may be configured programmatically or declaratively in the configuration file.

Configuring PathAttribute programmatically

The following example shows the creation of a new emitter which is configured with its HtmlWriter property set to an instance of ExternalFileWriter which has the PathAttribute property set to "href":

It should be remembered that the default TagName is "script" and that default FileExtension is "js". 

  IEmitter emitter = new StandardEmitter
                         {
                             HtmlWriter = new ExternalFileWriter
                                              {
                                                  PathAttribute = "href"
                                              }
                         }

Declaring PathAttribute in the config file

The following example shows the same declaration as above, as declared in a config file. 

    <section name="superScript" type="SuperScript.Configuration.SuperScriptConfig, SuperScript.Common" />
    <section name="superScript.ExternalFile" type="SuperScript.ExternalFile.Configuration.ExternalFileConfig, SuperScript.ExternalFile" />
    …
    <superScript>
      <emitters>
        <emitter key="externalJavascript">
          <writers>
            <modifier type="SuperScript.ExternalFile.Modifiers.Writers.ExternalFileWriter, SuperScript.ExternalFile">
              <properties>
                <property name="PathAttribute" value="href" type="System.String, mscorlib" />
              </properties>
            </modifier>
          </writers>
        </emitter>
      </emitters>
    </superScript>
