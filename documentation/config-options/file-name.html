FileName option

The FileName option allows setting the name of the returned file. 

  public string FileName { get; set; }

If not set, SuperScript will use an MD5 hash of the file's contents as the filename.

The getter for FileName will not return the hash which may later be used; the getter will only return an explicitly set value.

Configuring FileName

The FileName option may be configured programmatically or declaratively in the configuration file.

Configuring FileName programmatically

The following example shows the creation of a new emitter which is configured with its HtmlWriter property set to an instance of ExternalFileWriter which has the FileName property set to "user-details":

It should be remembered that the default file extension is "js". 

  IEmitter emitter = new StandardEmitter
                         {
                             HtmlWriter = new ExternalFileWriter
                                              {
                                                  FileName = "user-details"
                                              }
                         }

Declaring FileName in the config file

The following example shows the same declaration as above, as declared in a config file. 

    <section name="superScript" type="SuperScript.Configuration.SuperScriptConfig, SuperScript.Common" />
    <section name="superScript.ExternalFile" type="SuperScript.ExternalFile.Configuration.ExternalFileConfig, SuperScript.ExternalFile" />
    …
    <superScript>
      <emitters>
        <emitter key="externalJavascript">
          <writers>
            <modifier type="SuperScript.ExternalFile.Modifiers.Writers.ExternalFileWriter, SuperScript.ExternalFile">
              <properties>
                <property name="FileName" value="user-details" type="System.String, mscorlib" />
              </properties>
            </modifier>
          </writers>
        </emitter>
      </emitters>
    </superScript>

Both the above examples will emit the following result on the HTML page at the location where Emit.For("externalJavascript") is called: 

  <script type="/files/user-details.js" type="text/javascript"<>/script>
