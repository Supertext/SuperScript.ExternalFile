TagAttributes option

Gets or sets a collection of key-value pairs which form the tag's attribute collection. 

  public virtual IEnumerable<KeyValuePair<string, string>> TagAttributes { get; set; }

The default value is type="text/javascript>".

Configuring TagAttributes

The following example shows the creation of a new emitter which is configured with its HtmlWriter property set to an instance of ExternalFileWriter which has the TagAttributes property set to text="application/xml": 

  IEmitter emitter = new StandardEmitter
                         {
                             HtmlWriter = new ExternalFileWriter
                                              {
                                                TagAttributes = new List<KeyValuePair<string, string>>
                                                                    {
                                                                        new KeyValuePair<string, string>("type", "application/xml")
                                                                    }
                                              }
                         }

Unlike other properties on the ExternalFileWriter, the TagAttributes property cannot be set in the config file.
