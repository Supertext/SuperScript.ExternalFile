TagName option

Gets or sets the name of the enclosing HTML tag. 

  public virtual string TagName { get; set; }

The default value is script.

If the TagName is null or an empty string then not HTML tag will be emitted on the webpage.

Configuring TagName

The TagName option may be configured programmatically or declaratively in the configuration file.

Configuring TagName programmatically

The following example shows the creation of a new emitter which is configured with its HtmlWriter property set to an instance of ExternalFileWriter which has the TagName property set to "link": 

  IEmitter emitter = new StandardEmitter
                         {
                             HtmlWriter = new ExternalFileWriter
                                              {
                                                  TagName = "link"
                                              }
                         }

Declaring TagName in the config file

The following example shows the same declaration as above, as declared in a config file. 

    <section name="superScript" type="SuperScript.Configuration.SuperScriptConfig, SuperScript.Common" />
    <section name="superScript.ExternalFile" type="SuperScript.ExternalFile.Configuration.ExternalFileConfig, SuperScript.ExternalFile" />
    …
    <superScript>
      <emitters>
        <emitter key="externalJavascript">
          <writers>
            <modifier type="SuperScript.ExternalFile.Modifiers.Writers.ExternalFileWriter, SuperScript.ExternalFile">
              <properties>
                <property name="TagName" value="link" type="System.String, mscorlib" />
              </properties>
            </modifier>
          </writers>
        </emitter>
      </emitters>
    </superScript>
