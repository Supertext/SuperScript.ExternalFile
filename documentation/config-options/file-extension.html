FileExtension option

The FileExtension option allows setting of the file extension. 

  public string FileExtension { get; set; }

The default value is "js".

Configuring FileExtension

The FileExtension option may be configured programmatically or declaratively in the configuration file.

Configuring FileExtension programmatically

The following example shows the creation of a new emitter which is configured with its HtmlWriter property set to an instance of ExternalFileWriter which has the FileExtension property set to "xml": 

  IEmitter emitter = new StandardEmitter
                         {
                             HtmlWriter = new ExternalFileWriter
                                              {
                                                  FileExtension = "xml"
                                              }
                         }

Declaring FileExtension in the config file

The following example shows the same declaration as above, as declared in a config file. 

  <configuration>
    <configSections>
      <section name="superScript" type="SuperScript.Configuration.SuperScriptConfig, SuperScript.Common" />
      <section name="superScript.ExternalFile" type="SuperScript.ExternalFile.Configuration.ExternalFileConfig, SuperScript.ExternalFile" />
    </configSections>
    …
    <superScript>
      <emitters>
        <emitter key="externalJavascript">
          <writers>
            <modifier type="SuperScript.ExternalFile.Modifiers.Writers.ExternalFileWriter, SuperScript.ExternalFile">
              <properties>
                <property name="FileExtension" value="xml" type="System.String, mscorlib" />
              </properties>
            </modifier>
          </writers>
        </emitter>
      </emitters>
    </superScript>
  </configuration>

Both the above examples will emit the following result on the HTML page at the location where Emit.For("externalJavascript") is called: 

  <script type="/files/[file content hash].xml"></script>
