MakeUnique option

Gets or sets whether a hash of the contents should be appended as a query-string. 

  public bool MakeUnique { get; set; }

If no FileName is specified then a hash will be used for the FileName; If the hash is used as the FileName then it will not be appended as a query-string.

The default value is true.

The following table may help to explain the relationship between FileName and MakeUnique.

FileName MakeUnique Example output filename
null true mJzHRJsEFTFkeOO4CXUfaQ.js
"user-details" true user-details.js?h=mJzHRJsEFTFkeOO4CXUfaQ
null false mJzHRJsEFTFkeOO4CXUfaQ.js
"user-details" false user-details.js

The last example in the above table shows that if a filename is explicitly set and MakeUnique is set to false, a file's contents may be overwritten by a later request. This situation may or may not cause a problem.

Configuring MakeUnique

The MakeUnique option may be configured programmatically or declaratively in the configuration file.

Configuring MakeUnique programmatically

The following example shows the creation of a new emitter which is configured with its HtmlWriter property set to an instance of ExternalFileWriter which has the MakeUnique property set to false: 

  IEmitter emitter = new StandardEmitter
                         {
                             HtmlWriter = new ExternalFileWriter
                                              {
                                                  MakeUnique = false
                                              }
                         }

Declaring MakeUnique in the config file

The following example shows the same declaration as above, as declared in a config file. 

  <configuration>
    </configSections>
      <section name="superScript" type="SuperScript.Configuration.SuperScriptConfig, SuperScript.Common" />
      <section name="superScript.ExternalFile" type="SuperScript.ExternalFile.Configuration.ExternalFileConfig, SuperScript.ExternalFile" />
    </configSections>
    …
    <superScript>
      <emitters>
        <emitter key="externalJavascript">
          <writers>
            <modifier type="SuperScript.ExternalFile.Modifiers.Writers.ExternalFileWriter, SuperScript.ExternalFile">
              <properties>
                <property name="MakeUnique" value="false" type="System.Boolean, mscorlib" />
              </properties>
            </modifier>
          </writers>
        </emitter>
      </emitters>
    </superScript>
  </configuration>
