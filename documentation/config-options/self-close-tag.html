SelfCloseTag option

Gets or sets whether the output HTML tag is self-closing or has a closing tag. This property is only considered if TagName is a non-zero-length string. 

  public bool SelfCloseTag { get; set; }

The default value is false.

Configuring SelfCloseTag

The SelfCloseTag option may be configured programmatically or declaratively in the configuration file.

Configuring SelfCloseTag programmatically

The following example shows the creation of a new emitter which is configured with its HtmlWriter property set to an instance of ExternalFileWriter which has the SelfCloseTag property set to false:

It should be remembered that the default TagName is "script" and that default FileExtension is "js". 

  IEmitter emitter = new StandardEmitter
                         {
                             HtmlWriter = new ExternalFileWriter
                                              {
                                                  SelfCloseTag = true
                                              }
                         }

Declaring SelfCloseTag in the config file

The following example shows the same declaration as above, as declared in a config file. 

    <section name="superScript" type="SuperScript.Configuration.SuperScriptConfig, SuperScript.Common" />
    <section name="superScript.ExternalFile" type="SuperScript.ExternalFile.Configuration.ExternalFileConfig, SuperScript.ExternalFile" />
    …
    <superScript>
      <emitters>
        <emitter key="externalJavascript">
          <writers>
            <modifier type="SuperScript.ExternalFile.Modifiers.Writers.ExternalFileWriter, SuperScript.ExternalFile">
              <properties>
                <property name="SelfCloseTag" value="true" type="System.Boolean, mscorlib" />
              </properties>
            </modifier>
          </writers>
        </emitter>
      </emitters>
    </superScript>
