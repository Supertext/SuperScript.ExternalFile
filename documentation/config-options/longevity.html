Longevity option

The Longevity option allows setting of the lifetime of the file's contents. 

  public Longevity Longevity { get; set; }

The value is set by the Longevity enumeration. 

  public enum Longevity
  {
      /// 
      /// Indicates that this file should be deleted after first request.
      /// 
      SingleUse,

      /// 
      /// Indicates that this file should exist until further notice.
      /// 
      Reuse
  }

The default value is Longevity.SingleUse.

Important: If set to Longevity.Reuse, SuperScript.ExternalFile will not delete the contents of the file from the store. Such files must be deleted using the administration interface.

Automatic deletion of Longevity.Reuse files will be supported in a future release of SuperScript.ExternalFile.

Configuring Longevity

The Longevity option may be configured programmatically or declaratively in the configuration file.

Configuring Longevity programmatically

The following example shows the creation of a new emitter which is configured with its HtmlWriter property set to an instance of ExternalFileWriter which has the Longevity property set to Longevity.Reuse: 

  IEmitter emitter = new StandardEmitter
                         {
                             HtmlWriter = new ExternalFileWriter
                                              {
                                                  Longevity = Longevity.Reuse
                                              }
                         }

Declaring Longevity in the config file

The following example shows the same declaration as above, as declared in a config file. 

    <section name="superScript" type="SuperScript.Configuration.SuperScriptConfig, SuperScript.Common" />
    <section name="superScript.ExternalFile" type="SuperScript.ExternalFile.Configuration.ExternalFileConfig, SuperScript.ExternalFile" />
    …
    …
    <superScript>
      <emitters>
        <emitter key="externalJavascript">
          <writers>
            <modifier type="SuperScript.ExternalFile.Modifiers.Writers.ExternalFileWriter, SuperScript.ExternalFile">
              <properties>
                <property name="Longevity" value="Reuse" type="SuperScript.Longevity, SuperScript.ExternalFile" />
              </properties>
            </modifier>
          </writers>
        </emitter>
      </emitters>
    </superScript>
